# Data Analyst Portfolio

This bundle includes a modern portfolio site using **Bootstrap 5, jQuery, Font Awesome, AOS, Animate.css, and Google Fonts**.

## Files
- `index.html` — main page
- `style.css` — theme (colors, layout, components)
- `main.js` — interactivity (smooth scroll, counters, filters, modal, preloader, theme toggle)

## Customize
Search for placeholders in ALL CAPS (e.g., `YOUR NAME`, `TAGLINE`, social links, and Open Graph/JSON-LD URLs).

## Preview locally
Open `index.html` directly in a browser. For best results (and to avoid CORS issues later), use a simple server:
- Python: `python -m http.server 5500`
- Node: `npx serve`

Then visit `http://localhost:5500`.

## Deploy
- GitHub Pages / Netlify / Vercel: drag & drop these three files (or a folder) and publish.
